# BI Dashboards Learning Projects

This repository contains learning projects for data analysis and dashboard creation using Excel and Tableau.

## Purpose
- Practice data analysis and KPI visualization
- Understand dashboard design principles
- Build personal reference materials for BI tasks

## Included Projects

### Excel Dashboards
- Interactive dashboards created in Excel
- KPI calculations, charts, and metrics
- Data preparation and aggregation
  - [Dashboard 1](https://public.tableau.com/app/profile/tereshchuk.pavlo/vizzes)

### Tableau Dashboards
- Two dashboards created after online courses
- Practiced visualization, KPIs, and dashboard design
- Inspired by tutorial/course examples and extended for learning
- Accessible online via Tableau Public:
  - [Dashboard 1](https://public.tableau.com/app/profile/tereshchuk.pavlo/vizzes)
  - [Dashboard 2](https://public.tableau.com/app/profile/tereshchuk.pavlo/vizzes)
  
## Notes
All projects are tutorial/course-based learning projects created for educational purposes. They demonstrate practical skills in Excel and Tableau for data analysis and visualization.
